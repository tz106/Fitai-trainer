# FitAI Trainer

Proiect final Building AI Course  
Creat de *Tiberiu*

## Summary

FitAI Trainer este o aplicație bazată pe inteligență artificială care ajută utilizatorii să își atingă obiectivele fitness. Sistemul oferă planuri personalizate de antrenament, recomandări nutriționale și monitorizarea progresului.

## Background

Mulți oameni vor să facă sport și să ducă un stil de viață sănătos, dar nu știu ce exerciții să facă sau cum să înceapă. De asemenea, antrenorii personali pot fi costisitori.

Această problemă este foarte comună deoarece multe persoane nu au timp sau bani pentru sală și antrenori personali.

Motivația mea personală este că sportul este important pentru sănătate, iar tehnologia poate ajuta oamenii să se antreneze mai eficient.

## How is it used?

Utilizatorul introduce:

- vârsta
- greutatea
- înălțimea
- nivelul de experiență
- obiectivul (slăbire, masă musculară, întreținere)

AI-ul generează:

- plan de antrenament zilnic
- exerciții potrivite
- recomandări alimentare
- monitorizarea progresului

Poate fi folosit acasă, la sală sau oriunde.

## Data sources and AI methods

Date folosite:

- informații introduse de utilizator
- baze de date cu exerciții fitness
- date despre calorii și nutriție

Metode AI:

- machine learning
- recommendation systems
- predictive analytics

## Challenges

Proiectul nu poate înlocui complet un antrenor profesionist sau un medic.

Limitări:

- utilizatorii pot introduce date greșite
- fiecare corp reacționează diferit
- unele probleme medicale necesită specialist

## What next?

În viitor aplicația poate include:

- recunoaștere video pentru corectarea exercițiilor
- integrare cu smartwatch
- chatbot motivațional
- planuri avansate pentru sportivi
